# Pizza Store 🍕

## Explore Deployed Version

Explore the deployed version of Pizza Store [here](https://react-pizza-store-dev.vercel.app/).

## Overview

Pizza Store is an online pizza ordering platform built using React, TypeScript, Vite, CSS Modules, Redux, and Redux
Toolkit. It features the ability to switch between light and dark themes, as well as change the language between Russian
and English.

## Quick Start

1. **Clone the repository:**
    ```bash
    git clone https://github.com/devenrgy/react-pizza-store.git
    ```

2. **Install dependencies:**
    ```bash
    cd react-pizza-store
    npm install
    ```

3. **Run the development server:**
    ```bash
    npm run dev
    ```

   Visit [http://localhost:3000](http://localhost:3000) to view the application.

## Technologies

- **React:** JavaScript library for building user interfaces.
- **React Router:** Declarative routing for React applications.
- **TypeScript:** Typed superset of JavaScript that compiles to plain JavaScript.
- **Vite:** Fast, opinionated web development build tool.
- **CSS Modules:** Locally scoped styling for components.
- **Redux:** State management library for predictable state containers.
- **Redux Toolkit:** Set of tools for efficient Redux development.
- **i18next:** Internationalization framework for web applications.

## Project Structure

- **src/components:** React components used throughout the application.
- **src/styles:** Global styles and CSS modules for component styling.
- **src/pages:** Different pages/routes for the application.
- **src/api:** Handles API calls and data fetching.
- **src/utils:** Utility functions and helper modules.
- **src/assets:** Store static assets like images and icons.
- **src/store:** Manages state using a Redux-based store.
- **src/routes:** Configuration files for defining application routes.
- **src/i18n:** Localization files for language translation.

## Features

- **Pizza Selection:** Browse a variety of pizzas available for order.
- **Customization:** Customize pizzas with different toppings and crust options.
- **Order Placement:** Place orders and track delivery status.
- **User Authentication:** Sign in to manage orders and preferences.
- **Theme Switching:** Toggle between light and dark themes.
- **Language Switching:** Change the language between Russian and English.
